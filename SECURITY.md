# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |

## Reporting a Vulnerability

If you discover a security vulnerability in Office365LinuxInstaller:

1. **DO NOT** open a public GitHub issue.
2. Email the maintainer directly at the repository owner's contact (see GitHub profile).
3. Include:
   - A clear description of the vulnerability.
   - Steps to reproduce.
   - Potential impact.
   - Suggested mitigation (if any).

## Scope

This installer is a **deployment helper**, not a security product. It does not:
- Store credentials.
- Modify Microsoft Office binaries.
- Provide activation bypasses.

Reported issues related to Wine vulnerabilities or Microsoft Office security should be directed to their respective upstream projects.
